# DIO-Simulador-Investimentos-FIIs
Simulador de Investimentos em Fundos Imobiliários

Este repositório tem como objetivo aplicar os conceitos de Excel no desenvolvimento de uma ferramenta prática de simulação de investimentos em fundos imobiliários.

Material desenvolvido para fins didáticos, as simulações não tem objetivo de prever qualquer retorno sobre investimentos em fundos imobiliários.

# Conteúdo
- Planilha em formato .xlsx
- Informações necessárias: _salário_, _perfil do investidor_, _quantos anos pretende investir_ e _aporte mensal_.
- Planilha dinâmica onde as entradas de dados podem ser alteradas a qualquer momento pelo investidor.
- Os cálculos são executados a cada entrada de dados e os gráficos se ajustam automaticamente, sem botão de comando para calcular.
- Não existe uma ordem definida para inserção dos dados, porém cada dado alterado já será recalculado automaticamente.

# Observação
Os retornos estimados pela planilha são calculados utilizando o histórico de rentabilidade medido pelo IFIX (índice de fundo imobiliários), que reflete tanto a valorização das cotas quanto os dividendos pagos.

Como o IFIX foi criado em 2010, não existem dados consolidados para períodos maiores que 15 anos, levando isso em consideração o retorno acima de 15 anos mantém-se constante nos cálculos.

A planilha porém calcula até 40 anos de investimentos com objetivo de mostrar aos investidores o poder dos juros compostos e incentivar um pensamento de longo prazo.

# Como usar

Baixe a a planilha.

Preencha os campos indicados com as setas azuis.

Primeiramente, informe seus rendimentos, seu perfil de investidor e por quanto tempo deseja investir.			

A planilha então vai trazer uma segestão de aporte mensal.

<img width="831" height="287" alt="image" src="https://github.com/user-attachments/assets/2702f4fa-fd62-47cb-a3e6-991715a47ff9" />

---

Porém o investidor deverá informar o valor do aporte mensal manualmente, isso traz liberdade para simular outros valores não ficando preso ao valor sugerido.

Juntamente com o perfil de investidor informado na etapa anterior, será sugerido um percentual de investimento para cada tipo de fundo imobiliário, será informado também o valor que esse percentual representa.
